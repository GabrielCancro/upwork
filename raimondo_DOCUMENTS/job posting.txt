Godot Simple Demo

I have developed a simple demo game starting from a pre-built template.
Now I would to integrate some parts of this game with 2 sub-games.
I mean: in certain conditions, the user "discover" a subgame and can return to the main timeline after the game is ended.

The first game is to fill a table with "X" and only if the cells are filled in the right sequence a button become clickable and a foto will be showed.
Next, the user can continue to the main timeline.

The second game is a again a fill-to-check game: a user must discover a name by an anagram. If the user fill the right name, a button become clickable and a foto will be showed.

Both games must be inserted in the main timeline and a counter must be implemented.

The game is a demo and all raw graphics and video will be provided.

Please ask further if needed.
